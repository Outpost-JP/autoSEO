# autoSEO

## 概要
`autoSEO`は、SEO記事を自動生成するための試験的なプロジェクトです。このリポジトリは試作品であり、実現段階に至らなかったため現在は公開されています。

## 主な機能
- キーワード分析
- 記事の自動生成
- 記事の最適化


## インストール
1. リポジトリをクローンします。
    ```bash
    git clone https://github.com/caprice1026-disc/autoSEO.git
    ```
2. 必要な依存関係をインストールします。
    ```bash
    cd autoSEO
    pip install -r requirements.txt
    ```
